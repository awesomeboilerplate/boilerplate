Grid

GRID allows for the quick creation of column-based grid that can wrap or nowrap. It is lighter than the FEATURES or TILES mixins in that its contents have no default styling.

Emmet shorthand

.grid>(div>p)*3

Mixin usage

                                
div {
    @include grid($align: stretch, $columns: 3, $flow: row nowrap, $gutter: 1rem, $justify: space-between);
}
