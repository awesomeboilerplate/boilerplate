Select Wrapper

The colored-select-wrapper component allows you to style different select elements by passing a few parameters to set the border color, the accent color of the FontAwesome drop down arrow, and optionally, the height of the select and the border-radius. 

.select-wrapper {
    @include colored-select-wrapper($border, #FFF, 37px);
}
.some-other-class {
    @include colored-select-wrapper(#FFF, $brand-blue, 42px, 2px);
}

Emmet shorthand 

div.select-wrapper>select
