Modal

This modal component uses Magnific Popup, a carousel library by Dmitry Semenov. Magnific is extremely configurable, supporting different content types such as images, iframe, and html, and providing many options.

Image Popup Example

This is one of the simplest uses of magnific - zooming/expanding an image. Wrap the img in an a tag with the appropriate class and image source for the href, and use the title attribute as the image caption, as shown below:

Emmet:

a.image-popup[title="Image Caption"]>img[src="/path-to-image" alt="image alt text"]

   

Video Popup Example

This is a simple video popup. Wrap a video placeholder image in an a tag with the appropriate class, and the video link as the href:

Emmet:

a.video-popup[href="/path-to-video"]>img[src="/path-to-placeholder-image" alt="Video Placeholder Alt Text"]

HTML/Inline Popup Example(s)

This is a simple content popup which can include anything within a hidden div. Give your a tag the appropriate class, and set the div id as the href:

Emmet:

a.inline-popup[href="#id-of-hidden-div"]

Click Me!

This is an alternative example using the data-mfp-src attribute in an a tag instead of using the href.
Note: you can also use a span tag and style it like a button.

Emmet:

a.inline-popup[data-mfp-src="#id-of-hidden-div"]
