Search and Filters Component(s)

This is two components in one: a search listing that includes a simple search box and basic filters. See below for details.

Search Listing

The SEARCH LISTING component provides a starting point for a basic search results page..

Emmet for listing only, does not include search-filters:

div.search.listing>ul>li>(h2>a)+a.meta+div.summary>p

Search-Filters Component

The SEARCH-FILTERS component is a basic implementation of the button style filters using hidden checkbox inputs with adjacent span tags. Recommended use of wrapping in div with class "filters".

Emmet:

form.search-filters>(div.search>(div.text>label+input[type="search"])+(div.submit>input[type="submit"]))+(div.filters>(label>input[type="checkbox"]+span{filter name})*3)
